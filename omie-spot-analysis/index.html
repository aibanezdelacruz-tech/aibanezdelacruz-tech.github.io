📊 OMIE Spot Price Analysis

Análisis completo de precios horarios del mercado diario español (2023–2025)

📈 Resultados

Modelo MAE (€/MWh) RMSE (€/MWh) Mejora
Baseline (persistencia lag-24h) 18.33 26.90 —
XGBoost + ESIOS 6.91 10.60 ↓ 62.3%
Reducción de error: 62.3% (de 18.33 a 6.91 EUR€/MWh)
Dataset: 24,096 horas (2023-2025) Train: 19,000 horas Test: 5,000 horas (temporal split)

📁 Estructura del Proyecto

  • notebooks/ — Análisis interactivos (ejecutar en orden)
  • src/ — Módulos reutilizables (features, ESIOS client, loaders)
  • data/ — Raw data descargada desde OMIE y ESIOS
  • models/ — XGBoost modelo entrenado (.json, .pkl)
  • docs/ — Datos exportados para visualización web

🔬 Metodología

  • Ingesta de datos: Descarga directa OMIE (precio diario) + ESIOS REE (generación, demanda)
  • Feature Engineering: Lags, rolling stats, features de mercado, variables trigonométricas
  • Modelado: XGBoost con validación temporal (sin data leakage)
  • Validación: Coherencia energética verificada ✅
  • Análisis avanzado: Comportamiento en precios negativos, curtailment solar

🛠️ Tech Stack

  • Data: Python, pandas, numpy
  • ML: XGBoost, scikit-learn
  • Viz: Plotly, matplotlib
  • APIs: OMIE REST, ESIOS (REE)
